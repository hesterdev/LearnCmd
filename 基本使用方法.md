#DOS命令格式
##小巧的命令
1. 返回上级目录

    cd ..

- 返回到盘符

    cd \
    
- 该命令的参数及相关功能解释
    
    /?

- 查看文件夹内的树形结构
    
    tree
    
- 修改时间和日期
    
    date time

- 查看目录信息

    dir
   
- 复制(其中第一个参数可以是相对路径或绝对路径
    
    copy *.dll c:/user/dll 
    
##执行时可以不加文件扩展名(优先级 com>exe>bat)

- exe（可执行文件） 

- com（命令文件）

- bat(批处理文件）
--将若干个com/exe组合在一起，一次执行


##对一个目录而言，其还伴随着两个这样的目录
1. 当前目录

    .
    xcopy . G:
    xcopy . G: /s
- 上一级目录

    ..
        
- 根目录

    反斜杠 \  或 正斜杠 /

- 切换驱动器
    
    D:

- 删除路径
    
    del 路径  不删除子目录及其中的文件


## 文件夹的管理

1. 文件夹的创建

make directory

    mkdir 或  md


- 删除

remove directory

    rmdir 或 rd

## 目录管理

- 显示目录

    tree
    
- 显示文件和目录

    dir
    
- 改变当前目录

change directory

    chdir cd

- 删除目录树（废弃）

    deltree
    
## 文件夹重命名与路径设置

1. 设置搜索路径 - Path


2. 文件夹的重命名
    
    rename 或 ren
    
3. 文件复制
    
    copy
    
4. 文件的移动

    move
